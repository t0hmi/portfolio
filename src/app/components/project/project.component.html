Rock Paper Scissors

Angular
SCSS
In this project, I used Angular and SCSS to create a simple rock, paper, scissors game. The game allows the user to select their move and then compares it to the computer's randomly generated move. The result of the game is displayed on the screen, and the user has the option to play again. The design of the game is clean and modern, utilizing SCSS for styling. This project was a great opportunity to practice my Angular and SCSS skills.
